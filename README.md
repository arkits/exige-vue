# exige-vue

### Available Scripts

Setup project with  
`npm install -g @vue/cli`  
`npm install`

Run with Vue UI...  
`vue ui`   
or with npm   
`npm run serve`

Build  
`npm run build`

Test  
`npm run test`

Lints and fixes files  
`npm run lint`
